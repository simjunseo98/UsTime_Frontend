#UsTime-Project
커플 공유 캘린더 서비스 프로젝트

## 🖥️ 프로젝트 소개
-기존 커플 캘린더 공유 사이트의 복잡한 UI/UX와 불편한 일정 공유 방식을 누구나 쉽고 직관적으로 사용할 수 있도록<br> 개선을 목표로 하여 커플들을 위한 캘린더 서비스 및 일상 기록 사이트를 만들고자 기획하였습니다.
